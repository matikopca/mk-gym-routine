## How to Use

1. **Select a day**: Use the dropdown to choose Monday-Friday
2. **Add exercises**: Click "+ Add Exercise" to add new exercises
3. **Track progress**: Tap checkboxes to mark sets as complete
4. **Edit/Delete**: Hover over exercises to see edit/delete options
5. **Reset**: Use reset buttons to clear progress

## Exercise Types

- **Strength**: Weight training exercises
- **Cardio**: Cardiovascular exercises  
- **Flexibility**: Stretching and flexibility work

## Installation

Simply open `index.html` in any modern web browser. For mobile use:

1. Open the file in Safari on your iPhone
2. Tap the Share button
3. Select "Add to Home Screen"
4. Your gym tracker is now a web app!
